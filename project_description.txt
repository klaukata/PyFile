Creating a practical solution to simplify my daily tasks and highlight my Python skills, "PyFileOrganizer" automates
file management for a more organized digital life. This project is a testament to my proficiency in Python and a
commitment to efficiency in everyday workflows.

# INTUITION = what to do Upon receiving a user-specified folder path containing a diverse array of files, the program
autonomously creates a designated folder. Subsequently, it systematically organizes the assorted files within this
folder based on their types.
